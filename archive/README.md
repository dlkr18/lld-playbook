# archive/

Historical, **non-authoritative** files kept only for reference. Nothing here is current
spec — do not rely on it when reasoning about the repo's present state.

- `status-reports/` — old one-off progress/completion reports (`*_COMPLETE.md`, `*_STATUS.md`,
  `*_FIX.md`, `GITHUB_PAGES_*.md`, …) from earlier build phases.
- `manifests/` — generator / subagent handoff notes (`MANIFEST_*.md`) describing what a given
  content batch produced.

Safe to delete wholesale if you don't need the history.
